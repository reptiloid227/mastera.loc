Автоматическое добавление правильных Alt и Title для изображений при использовании загрузчика WordPress.

== Description ==

Автоматическое добавление правильных Alt и Title для изображений при загрузке. Легкий код и грамотное заполнение метатегов изображений с точки зрения SEO, для Яндекса и Google.

Плагин изменяет работу встроенного загрузчика изображений. Стандартный загрузчик копирует название изображения в поле «Заголовок» (title), а поле «Атрибут alt» (alt) оставляет пустым. Плагин, в свою очередь, копирует название изображения в поле «Атрибут alt», а в поле «Заголовок» подставляет конструкцию: «Изображение — название изображения». Таким образом, соблюдаются рекомендации поисковых систем по представлению информации изображений без лишних манипуляций.

= Features =
* Простое изменение конструкции «Изображение — » на любое другое
* Удаление лишних символов (кавычек, дефисов, двойных пробелов и других символов) из названия изображения
* Соблюдение рекомендаций рекомендаций поисковых систем по представлению изображений
* Легкий код плагина
* [Имеет много преимуществ перед аналогичными плагинами](https://prostudio.ru/journal/prostudio-auto-meta-images/)

== Installation ==

1. Загрузите папку "prostudio-auto-meta-images" в директорию "/wp-content/plugins/".
2. Активируйте плагин на странице "Плагины" в панели управления WordPress.

== Frequently Asked Questions ==

= Как изменить конструкцию «Изображение — »? =

Зайдите в папку плагина и откройте файл "prostudio-auto meta-images.php", найдите следующий код:

`
$my_image_meta = [
		'ID' => $post_ID,
		'post_title' => 'Изображение — ' . $my_image_title, // Set image Title to sanitized title
]; 
`

Измените конструкцию «Изображение — », находящуюся в одинарных кавычках на любое другое (56 строчка, при просмотре в редакторе).
